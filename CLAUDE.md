# DoDroidAI

Agent App 项目，通过对话直接操作手机本身。

## 项目概述

这是一个 Android Agent 应用，用户可以通过自然语言对话直接操作手机，执行各种任务。

## 技术栈

- **语言**: Kotlin
- **UI**: XML Views
- **架构**: 待补充
- **Gradle**: 9.1.0
- **AGP**: 9.0.1
- **Kotlin**: 2.0.21

## 项目结构

```
app/
├── src/main/
│   ├── java/com/example/dodroidai/
│   │   ├── MainActivity.kt
│   │   └── data/
│   │       └── DataRepository.kt
│   └── res/
│       ├── layout/          # XML 布局
│       ├── drawable/         # 图标资源
│       ├── mipmap-*/        # 应用图标
│       └── values/           # 字符串、主题等
```
